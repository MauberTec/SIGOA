Módulo Integração

 

 

VDM

 

Primeiro item do Módulo Integração:

O DER através de um midleware disponibilizou acesso aos dados de volume diário médio de seu sistema VDM.

O valor do VDM expressa a quantidade média de veículos que utiliza o trecho por dia, estando  enquadrado dentro de faixas que ponderam a Importância da Obra de Arte na Malha Viária.

 

O IP ESTÁ DANDO ERRO!!!!

 

TPU

 

Segundo item do Módulo Integração:

O DER através de um midleware disponibilizou acesso aos dados da Tabela de Preços Uniários de seu sistema TPU.

O sistema TPU não contempla até o momento as composições de preços unitários de Reparos, item desenvolvido neste contrato.

Para permitir o orçamento dos serviços de recuperação indicados pelo sistema ou adotados pelo responsável pela inspeção especial, o SIGOA está cadastrado com os valores unitários das CPUs Maubertec cuja data base é de Julho de 2020.

Estas CPU's deverão ser incorporadas oportunamente ao sistema TPU do DER, para que as atualizações de preços sejam automáticas.

Quando isso ocorrer, o usuário deverá indicar na Tela Associação de Reparos a TPU a nova Fonte de preço (DER) e o código de cada TPU no sistema do DER.

Desta forma, o sistema poderá orçar seja através da tabela Desonerado ou Não Desonerado com a Data base selecionada, os reparos e serviços extraordinários que possam ser requeridos.

A obtenção do preço é feita automaticamente pelo sistema, entretanto, através do Módulo Integração, o usuário poderá fazer consultas aos dados do sistema TPU, e verificar que a integração está ocorrendo de maneira correta.

Na parte superior da tela TPU, são apresentados os filtros Ano, Fase, Mês e Desonerado, para permitir ao usuário restringir o número de registros que serão apresentados no grid abaixo.

O usuário poderá inserir filtros e clicar no botão  para visualizar apenas os registros que atendam os critérios estabelecidos.

Para limpar os filtros de forma a exibir todos os registros cadastrados, basta clicar no botão .

SIRGeo

 

Terceiro item do Módulo Integração:

O DER através de um midleware disponibilizou acesso aos dados tabulares do seu sistema SIRGEO.

O sistema SIRGEO é o integrador dos vários sistemas deste projeto, e é responsável pelo georeferenciamento das obras de arte e disponibilização de mapas temáticos.

No SIRGEO são disponibilizados informações da Regional responsável pela OAE que são utilizadas para envio dos alertas relativos aos serviços de conserva, e também para filtrar as informações dos relatórios gerenciais.

Por outro lado, o SIGOA disponibilizará através de MIDLEWARE os dados relativos à priorização, para que seja possível para o SIRGEO, criar um mapa temático com o ranqueamento das obras de arte.

Para cada OAE ranqueada, o SIGOA disponibilizará os dados de: Rodovia, Km inicial da OAE e Nota Geral, e no SIRGEO estariam configurados que para as OAES georeferenciadas e ranqueadas:

 

- Notas entre 15 e 11 - apresentadas na cor VERDE indicando um bom estado,

- Notas entre 10 e 6 - apresentadas na cor AMARELA, indicando um estado de atenção

- Notas entre 5 e 1 - apresentadas na cor VERMELHA indicando um estado preocupante.

Na parte superior da tela SIRGeo, é apresentado o filtro Rodovia, para permitir ao usuário restringir o número de registros que serão apresentados no grid abaixo.

Digitando o código de uma rodovia específica e clicando no botão  serão visualizados apenas os registros da rodovia selecionada.

Para limpar o filtro de forma a exibir todos os registros cadastrados, basta clicar no botão .
